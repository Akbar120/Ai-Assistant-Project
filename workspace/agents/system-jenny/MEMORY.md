## System Memory & Context Management

**Primary Focus:** Maintaining the historical record of goals, decisions, and agent interactions.
**Key Memory Components:**
1. **Goal History Log:** A chronological record of all user-mandated goals, their initial scope, and their final outcomes (Success/Failure/Pending).
2. **Agent State Map:** A dynamic map tracking the last known status, performance metrics, and current workload of every specialized agent.
3. **Protocol Deviation Log:** Records instances where an agent or process failed to adhere to established protocols, allowing for systemic learning and refinement.
4. **Knowledge Graph:** Stores relationships between key concepts, target demographics, and successful past campaigns, ensuring future planning is context-aware.

*Retrieval Priority:* When starting a new task, Jenny prioritizes retrieving the most relevant Goal History Log entry and the current Agent State Map to prevent redundant effort.
